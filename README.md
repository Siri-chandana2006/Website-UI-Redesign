# Red Bull Website UI Redesign

## Project Overview

This project redesigns the Red Bull website to improve usability, visual hierarchy, and responsiveness across desktop and mobile devices.

## Objectives

* Perform heuristic evaluation
* Create a mood board and style guide
* Design high-fidelity mockups
* Ensure responsive layouts

## Deliverables

* Desktop Homepage Design
* Mobile Homepage Design
* Mood Board
* Style Guide
* Heuristic Evaluation Report

## Tools Used

* Figma
* GitHub

## Outcome

A modern and responsive redesign that improves user experience, navigation, and visual consistency.
