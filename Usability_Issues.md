# Heuristic Evaluation

## Website Chosen

Red Bull

## Issues Identified

1. Homepage contains a large amount of content which can overwhelm users.
2. Navigation could be simplified for quicker access to key sections.
3. Visual hierarchy is inconsistent across some sections.
4. Mobile layouts can be optimized for better readability.
5. Calls-to-action are not always prominent.

## Improvements Implemented

1. Simplified navigation structure.
2. Improved visual hierarchy with better spacing and typography.
3. Added clear call-to-action buttons.
4. Created responsive desktop and mobile layouts.
5. Organized content into structured sections for better usability.
